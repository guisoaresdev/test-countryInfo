# Backend

Install Dependencies: 
````
npm install
````

Setup Environment Variables:
````
cp .env.example .env
````

Commands:
````
npm start (Run server)
npm run lint (Run ESLint)
npm run format (Run Prettier)
````

# Frontend

Install Dependencies:
````
npm install
````
Setup Environment Variables:
````
cp .env.example .env
````

Commands:
````
npm run build (Build project)
npm start (Run Project)
npm run lint (Run Next Lint)
````
